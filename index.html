SummaPDF

Résumé intelligent et traduction locale de documents PDF. 100% gratuit et privé.

Avertissement : L'application est ouverte localement via le protocole file://. Certains navigateurs bloquent les Web Workers locaux. Si l'extraction ou le résumé ne se lancent pas, veuillez héberger le dossier sur GitHub Pages ou exécuter un serveur local (ex: extension VS Code Live Server ou python -m http.server).
